macOS を MacOS とか OS X とか書く知的生命体を許せないタイプのもみにすです。
ある日突然、父親から MacBook Pro をもらいました。そこにドラマは一切ないので省略。
とりあえずさくっと macOS ファーストインプレッションを書きます。**結論から言うとアンチやめます**。

## 操作性がすごい

私はこの OS の操作性に驚くべき快適さを発見したが、文字に書くには語彙が狭すぎる。

フルスクリーンをじゃんじゃん使ってくのがかっこいい。「macOS はウィンドウのツールバーとタスクバーが合体する」とは聞いていたものの、やっぱり実際に触ってみるのが一番だと思う。
Ubuntu 上の GNOME にプラグインを入れまくって最大化時にタイトルバーを消したりなんだりの設定をしてたのを思い出した。そういえば ElementaryOS なんて Ubuntu 系 OS もあったな、と回顧する。

macOS の最大化は Windows のそれとは違ってタイトルバーが消失してしまい、本当にフルスクリーンで表示しているようになる。そうするとやっぱり画面が広く使えていい気分だ。
しかもフルスクリーンにしたアプリは一つの新しい仮想デスクトップのような挙動をとるので、必要なアプリが全画面表示の下に隠れてどっかいったということもない。

それでいてアプリ同士はトラックパッドで指 3 本横スワイプを撃てば恐ろしいほど滑らかに切り替えることができる。
このトラックパッドが本当に優秀で、頬ずりしたくなるほどすべすべで手触りのよい材質をしている。あまりの美しさに意味もなくアプリ切り替えまくって反復横跳びを繰り返すことはもはや必然であった。

あとはショートカットの操作体系がだいぶ統一されていて便利。どのアプリでも Command+Comma を押せば環境設定が開いてくれる。Command と Control が分離されてて、ターミナルを使う時にコピーの Cmd+C と SIGINT を送る Ctrl+C を分離できるのもかなり高得点。

標準 IME もそこそこ賢く、スペースを叩かないでも勝手に変換してくれる機能(ライブ変換)が便利で、まあスペースを叩かないでも変換できるので結構な速さで文字が書ける。ただこれを使っていると漢字をひらくのが難しいなあとは考えた。もしかしたら Google 日本語入力とかでもあったりするのかもしれない。だけども Google 日本語入力で使えてた三点リーダとかが打てないのはなんだかなあ…。Windows/macOS 両対応の OSS 日本語入力ソフトが欲しいので作りたいとちょっと思っている。だが大変そうだ。

## ああ…う、美しすぎます！

Windows は汚くて macOS は綺麗なものといったらなんだろうか？当然見た目の綺麗さである。特にフォント。たしかに MacBook の文字はバカみたいに綺麗で、外部ディスプレイで使っている Windows の文字とは雲泥の差である。

Windows の文字が汚いと言われるのはいろいろ理由があると思っていて、ざっと挙げれば

- 単純に ClearType のレンダリングが微妙
- MacBook が解像度の暴力を振り回してるだけ
- フォント自体(字体と自体をかけてある)がダサい
  - MS ゴシック
  - UI フォント

といったところか。一番上のレンダリング能力の差は正直そんなないような気もしている。普段使っているモニター(21.5 インチ, 1920x1080)に MacBook を繋いでみたが劇的にいいかというと…うーん…
正直なところよくわからん。なんか Windows より明るめに出ている気がするが色管理かディスプレイの設定かもしれない。macOS がサブピクセルを使わないせいか、小さい文字になるとむしろ汚くなっているとすら思う。でもやっぱり細かな設定のあれこれかもしれない。

そんなこんなで外部ディスプレイはそんな綺麗でもないのだが、MacBook 本体のディスプレイはバカみたいに綺麗な文字を出力している。これはひとえに解像度が高すぎるだけだろう。先述の通り普段使いのディスプレイは 1920x1080 の 21.5 インチというあまりに凡庸なものである。それに対してこの MacBook Pro のディスプレイはなんと**2560x1600**という超解像度が**12 インチ**に鮨詰めになっているのだ。こんな画素密度で描画したらなんだって綺麗に見えてしまうだろう。

あと Windows の場合古いソフトが MS ゴシックで文字を出すことが頻発するから汚いというのもある。ニコニコ動画を開くと様々な動画のテロップに MS UI Gothic を見出すことができると思うが、これもかの有名な AviUtl の初期設定フォントが MS UI Gothic になってるおかげである。システム UI のフォントが縦長なのも評判が悪いらしい。自分はそこまで悪いとも思わないのだが麻痺しているだけかも。とりあえず全人類は Word とパワポで MS ゴシックを使うのをやめろ。ついでに HG 創英角ポップ体を授業プリントで太字がわりに使うのもやめろ。

一方 macOS にはみんな大好きヒラギノや Osaka が入っており、クソダサい MS ゴシックを見る必要はない。なんとすばらしいことか。ただ Arial Unicode MS とかいう日本語フォントとしては 0 点ものの産廃がしれっと混ざっているのはどういうことなんだろう。

フォント以外も綺麗で、特に MacBook の高画質ディスプレイで見る壁紙は息を飲む美しさだった。ただ光沢液晶なので自分の顔が映るのがいただけない。ほんとイケメン専用で困る。

他にはウィンドウやメニューにしっかり角丸が入っているのもきれい。Windows の手に刺さったらめちゃくちゃ痛そうなやつとは違うのだ。

## まとめ

macOS はすごい。というか MacBook がめちゃくちゃすごい。ハードとソフトが合わさって最強に見えるのは iPhone/iOS, iPad/iPadOS と同じ Apple の十八番だし。Mac がすごいので Windows にももっと頑張って欲しい。Linux は知らん。開発者でない人間が(も)触るものが OSS であるメリットが少ないので一生繁栄しないと思う。彼らにはおとなしく Windows の標準シェルとして働いてもらいます。

## 補遺

1. Windows 版 VSCode の候補表示(Ctrl+Space)をうまく macOS でやる方法を募集中です。Command+Space は Spotlight 検索だし Ctrl+Space は IME 切り替えになってて無効化してもなぜか VSCode 側にキー情報が行かない。
1. MacBook のヒンジがグッというかんじでしっかり止まってくれるのがたのしい。さすが Apple だぜ。
1. 21 インチディスプレイと 5 インチスマホの画素数が同じなのちょっとおかしくない？って思うので、もっと 4K ディスプレイとか安くなってほしい。
