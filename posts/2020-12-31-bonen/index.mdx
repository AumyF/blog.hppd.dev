# プログラミングやってなくね？

イヤ，やった気はするしやったはずなんだが，なんかアウトプットが少ないというか，なにかソフトウェア作ったぜというのがぜんぜんできていない．カス．

今 GitHub の Contribution 読み返しているがほとんどブログだしな…いつまでブログいじくってんだこのアホと言われそうなのでこれをもってしばらくはやめます．というか Chakra UI への移行は完了したのでもうやることはないはず．

ちなみに OSS への contribute としては他には Gatsby のドキュメントの日本語訳に参加したり，prompts にバグ修正のプルリクがマージされたりした．実は prompts のアレが初めてのコード貢献だったりする．でもそれ以外ぜんぜんやってねえ．Gatsby の翻訳も結局ばくはつ炎上して立ち消えになったっぽいし…

まあ Gatsby が今年ってことは Elm をほんのちょっと触ったのも今年，React の理解がまあまあ深まったのも今年なので，今年は目に見えづらい学習を行ったということでここはひとつ勘弁していただきたい．

年末というかほんの 6 日前だが uenojs に参加させていただいたりもした．ああいうものはリアルでも仮想空間上でもほとんどやってこなかったので慣れてなくて大変だった．もちろん技術力のある人があつまってて面白いんだけど，自分の引き出しが少なくて逆に面白要素を提供できないと申し訳ない…申し訳ない…って気持ちになる．上記したがなにかプロダクトてきなのを作ったわけじゃないし，そもそもプロダクトを作るより型を書いてるほうが楽しいと思っちゃうタイプのようだ．IT エンジニアよりコンピュータ科学者のほうが向いてるんじゃないか？

ちなみに Qiita 記事はわりとウケたのがあって，それが [(今はまだ)WSL1 に Ubuntu 20.04 を入れるな](https://qiita.com/AumyF/items/eaf42dd3345a2285ff9e)．あれはわりと勉強になったと思う．2021 年はめっちゃバズる記事を (Zenn に) 書く．

じつはおーみー名義が生えたのは今年．へえそうなんだ．

# 勉強してなくね？

高校生ですがほとんど受験勉強とかしていません．というか高校受験もほぼ受験してないのと同じようなもんなのでいままで受験勉強とかまともにやったことがない．ゴミ．本当に大学進学希望してるのか？

あ，一応言っておくと 2003 年生まれの 2020 年度 2 年生なので，もうすぐ共通テストだけど勉強してませんワロス，とかではないです．一応あと 1 年あります．もちろんあと 1 年しかない，の誤りです．

正直言うとコンピュータ勉強するのに国語とか勉強するのめんどくさい．これについては，大学進学の段階に興味のある分野が決定してる人間がむしろ異常者であるとの意見も出されている (誰によって？)．コンピュータ力で殴れば勝手に合格したりしないかな～と思っているがそれに足るほどのコンピュータ力がないことは前節で述べたとおり！やばい！
